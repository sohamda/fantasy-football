## Project Overview

This project is a comprehensive Fantasy Football application designed for the Dutch Eredivisie league. It includes:

- **Player Data Extraction** - AI-powered extraction of player statistics from various sources
- **Sportmonks API Integration** - Real-time football statistics from the Sportmonks API
- **Points Calculator** - Position-aware fantasy point calculation engine
- **AI Agents** - Automated player analysis and recommendation agents
- **React Frontend** - Modern UI for team management and visualization
